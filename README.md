A command line successor to Xifon's MW2SA. Maps in game usernames to IP addresses so they can be kicked if you are host. Designed for use with Call of Duty Modern Warfare 2 (2009) and only Windows.

Resources and libraries used:
 1. Visual Studio 2026 (MSVC v145).
 2. [npcap](https://npcap.com/) for packet capture.
 3. [Ghidra](https://github.com/NationalSecurityAgency/ghidra) to reverse engineer game binaries.
 4. Jacek Fedoryński's fantastic write up on [Quake III Arena's network protocol](https://www.jfedor.org/quake3/).
